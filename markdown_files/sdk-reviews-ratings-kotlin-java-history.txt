# История обновлений

## Версия SDK 2.0.0​

Внутреннее обновление SDK.

## Версия SDK 1.0.0​

Внутреннее обновление SDK.

Внутреннее обновление SDK.

Используйте `launchReviewFlow(reviewInfo)`.
