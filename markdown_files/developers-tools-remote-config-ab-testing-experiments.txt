# A/B эксперименты в Remote config

Для просмотра результатов экспериментов и управлениям параметрами в [Remote
Config](https://remote-config.rustore.ru) доступен раздел «A/B эксперименты»:

  * Просмотрите список созданных экспериментов. Список удобно фильтровать по статусу или дате, или ввести название эксперимента в поле «Поиск».
  * Узнайте количество участников и результаты эксперимента.
  * Создайте новый или редактируйте созданный ранее эксперимент.
