# Как работать с метрикой

Вы можете редактировать, копировать и удалять созданные метрики.

предупреждение

Чтобы работать с метриками, подключите
[MyTracker](/help/developers/tools/remote-config/ab-testing/mytracker-
connection).

## Редактирование​

  1. Откройте [Remote Config](https://remote-config.rustore.ru).
  2. Выберите приложение.
  3. Перейдите во вкладку «Метрики».
  4. Найдите метрику в списке.
  5. В меню действий выберите «Редактировать».
  6. Внесите изменения.
  7. Нажмите «Сохранить».

## Копирование​

  1. Откройте [Remote Config](https://remote-config.rustore.ru).
  2. Выберите приложение.
  3. Перейдите во вкладку «Метрики».
  4. Найдите метрику в списке.
  5. В меню действий выберите «Копировать».

к сведению

На открывшейся странице появится копия существующей метрики с заполненными
данными. Отредактируйте их, если нужно.

  6. Нажмите «Сохранить».

## Удаление​

  1. Откройте [Remote Config](https://remote-config.rustore.ru).
  2. Выберите нужное приложение.
  3. Перейдите во вкладку «Метрики».
  4. Найдите метрику в списке.
  5. В меню действий выберите «Удалить».
  6. Подтвердите удаление.
