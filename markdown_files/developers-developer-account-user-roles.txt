# Управление доступами и ролями пользователей в RuStore Консоли

Вы можете установить ограничение возможных операций с приложениями для
сотрудников и назначить им определённые роли. Это позволит чётко настроить
варианты взаимодействия с приложениями. Например, разрешить сотруднику только
публиковать приложения или совершать возвраты.

## Как добавить пользователей​

  1. Выберите приложение.

  2. Перейдите на вкладку «Доступы» в меню слева.

  3. Нажмите кнопку «Пригласить пользователя» в верхней части экрана.

![](https://www.rustore.ru//help/assets/images/2e6dae7fbab03f48ba5997f5e800c42a-9c75906b00f91c1956096d1492f42f9f.png)

  4. Укажите VK ID пользователя. [Как узнать VK ID?](/help/developers/developer-account/your-vkid)

  5. Выберите роль пользователя. Можно выбрать только один вариант из списка.

![](https://www.rustore.ru//help/assets/images/user-roles-
fa9e041f089cccc20f522d2038ff729d.png)

  6. Нажмите кнопку «Пригласить».

![](https://www.rustore.ru//help/assets/images/a3bb6d46c63fa6f15de3ab4b6cada5ce-9d7fd2cdf1d044e89a253f68e963e96e.png)

## Как редактировать доступы​

  1. Выберите пользователя.

  2. Нажмите на ![icon](https://www.rustore.ru//help/pencil-icon.svg) рядом с ним.

  3. Выберите новую роль.

![img](https://www.rustore.ru//help/assets/images/edit-
role-1c915ce2752b54820445069cf3c57e27.webp)

  4. Нажмите кнопку «Сохранить».

## Как удалить пользователя​

  1. Выберите пользователя.

  2. Нажмите на ![icon](https://www.rustore.ru//help/cross-icon.svg) рядом с ним.

![](https://www.rustore.ru//help/assets/images/f6879866043f5e0c8a4eec6670834387-9b88da99e282fdff8ea6338e0181a09e.png)

  3. Нажмите кнопку «Удалить».

предупреждение

Будьте внимательны, отменить удаление не получится и придётся заново добавлять
пользователя в случае ошибочного удаления.

## Какие роли можно выбрать​

В RuStore возможны шесть ролей пользователей:

  * владелец компании;
  * администратор;
  * релиз-менеджер;
  * разработчик;
  * финансовый менеджер;
  * специалист поддержки.
