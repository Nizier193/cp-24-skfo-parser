# История изменений SDK

На этой странице представлены альфа-релизы SDK, которые позволяют
разработчикам одними из первых опробовать обновления RuStore SDK.

предупреждение

Во время альфа-сборок может меняться внешний интерфейс.

## Версия SDK 6.0.0​

**SDK обновлений**

  * Добавили блок «Что нового» в обновлении, которое предлагается установить пользователям.

**SDK платежей**

  * Добавили в ответ метода покупки продукта признак «тестовый платёж».
