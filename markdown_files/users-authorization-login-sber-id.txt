# Вход через Сбер ID в RuStore. Подробная инструкция с добавлением карт

Вы можете авторизоваться в RuStore через Сбер ID:

  1. Нажмите ![icon](https://www.rustore.ru//help/rustore-icon.svg) на домашнем экране вашего устройства.
  2. Нажмите «Войти».
  3. Нажмите ![icon](https://www.rustore.ru//help/sber-icon.svg), чтобы авторизоваться через приложение Сбербанк.
  4. Войдите в приложение Сбербанка, чтобы авторизоваться в RuStore.

## Как добавить карты из Сбер ID​

После авторизации вы можете разрешить привязку карт из Сбер ID в профиль
RuStore. Для этого:

  1. Поставьте галочку рядом с пунктом «Добавить к аккаунту ранее сохранённые карты в Сбер ID». Так вы подтвердите своё согласие.
  2. Нажмите кнопку «Продолжить».
  3. Карты из Сбер ID появятся в разделе «Способы оплаты». [Как редактировать данные карт?](/help/users/purchases-and-returns/sber-id-connect)
