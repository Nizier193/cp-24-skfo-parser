# Сертификат разработчика для ОС Аврора

Установочные пакеты приложений для ОС Аврора должны быть подписаны
сертификатами. Сертификаты идентифицируют поставщиков ПО и защищают мобильные
устройства от попадания нежелательных приложений.

Сертификат разработчика может быть выдан в единственном экземпляре для одного
верифицированного аккаунта разработчика в RuStore.

## Подача заявки на сертификат​

Чтобы подать заявку:

  1. Откройте [RuStore Консоль](https://console.rustore.ru/sign-in).
  2. Перейдите на вкладку «Компания».
  3. Выберите «Сертификат для ОС Аврора» в боковом меню.
  4. Нажмите «Подать заявку».

![](https://www.rustore.ru//help/assets/images/cert-41c7f80a986e67742f1680fe40707f8f.png)

В модальном окне вам необходимо:

  1. Загрузить файл с csr-запросом на сертификат, сгенерированным согласно [инструкции](/help/developers/publishing-and-verifying-apps/app-publication/app-publication-aurora/dev-certificate/generate-csr);

примечание

Загружаемый csr-файл может быть форматов .pem, .csr или .p10 и размером не
более 500 Кб.

  2. Указать e-mail адрес для получение сертификата и обратной связи;
  3. Ознакомиться и согласиться с условиями.

![](https://www.rustore.ru//help/assets/images/Modal-
certificate-7aa0ae626bb1d4ddbb738674df5cc285.png)

Кнопка «Подать заявку» станет активна после выполнения всех пунктов.

примечание

Если вы допустили ошибку при заполнении формы, вы можете отправить заявку
повторно. Количество принимаемых заявок ограничено.

После отправки заявки ожидайте ответа на указанный e-mail адрес. С вами могут
связаться для уточнения информации. После обработки заявки вы получите файл-
сертификат для подписи ваших приложений.
