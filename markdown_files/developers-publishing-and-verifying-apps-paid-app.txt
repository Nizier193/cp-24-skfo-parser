# Руководство по платным приложениям в RuStore

При добавлении нового приложения систему RuStore Консоль выберите тип
приложения: бесплатное или платное.

к сведению

Подключите монетизацию, чтобы загружать платные приложения. [Как это
сделать?](/help/developers/monetization/enable-monetization)

  1. Откройте систему [Консоль RuStore](https://console.rustore.ru/sign-in).

  2. Перейдите на вкладку «Приложения» и нажмите «Добавить приложение».

![](https://www.rustore.ru//help/assets/images/first_app-6ca7de998c48cbdb5a6ce4388370bd70.png)

  3. В новом окне выберите тип приложения: «Универсальный» или «Только для ТВ».

Выберите «Универсальный» тип, если APK вашего приложения подходит для
телефона, планшета и ТВ.

примечание

Для ТВ доступны только бесплатные приложения: если загружен apk-файл для ТВ,
создание платного универсального приложения невозможно.

  1. Выберите параметр «Платное».

предупреждение

Изменение параметра будет недоступно для этой версии приложения.

  2. Введите название приложения и нажмите «Добавить».

![img](https://www.rustore.ru//help/assets/images/paid-1f8dbdc05b769294222f0ad26b6a2a52.webp)

Вы можете обновить цены вашего приложения при редактировании или при [загрузке
новой версии](/help/developers/publishing-and-verifying-apps/app-
publication/new-version-app) приложения.

## Как изменить цену приложения​

  1. Откройте [Консоль RuStore](https://console.rustore.ru/sign-in).
  2. Перейдите на вкладку «Приложения».
  3. Выберите приложение.
  4. Нажмите «Редактировать».
  5. Введите новую стоимость приложения.
  6. Нажмите «Отправить на модерацию».

Перед отправкой приложения на модерацию обязательно проверьте приложение на
соответствие [требованиям к приложениям](/help/developers/publishing-and-
verifying-apps/requirement-apps).

Будет создана новая версия приложения, которая отправится на модерацию. После
прохождения модерации статус приложения изменится и появится кнопка
«Опубликовать».

Для публикации приложения нажмите «Опубликовать». Новая цена приложения
отобразится в RuStore.

![](https://www.rustore.ru//help/assets/images/paid_app_full-6484038889f8d33311e57c80aed2596a.png)

Чтобы настроить цены на товары в приложении, узнайте, [как создавать платный
товар в приложении](/help/developers/monetization/create-paid-product-in-
application).

Чтобы настроить цены на подписки, узнайте, [как создавать подписку на
приложение](/help/developers/monetization/create-app-subscription).

## Комиссия в платных приложениях​

Комиссия RuStore составляет 15% от дохода разработчика.

Актуальные условия по размеру комиссии можно посмотреть в разделе «[Соглашение
RuStore о распространении программных продуктов](/help/legal/developers-
agreement)».
