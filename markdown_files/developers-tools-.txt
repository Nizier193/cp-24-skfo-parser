# Оптимизация и управление приложениями через инструменты RuStore

Используйте раздел «Инструменты» для подключения инструментов разработчика к
вашим приложениям. Используя инструменты, вы получаете дополнительные
возможности:

  * вовлечение пользователей;
  * удаленное управление приложением;
  * использование аналитики;
  * поиск сбоев и анализ проблем;
  * упрощение и ускорение разработки за счёт интеграции SDK.

Подключение инструментов доступно только пользователям с правами владельца
организации.

## Подключение инструмента​

подсказка

В разделе «Инструменты» есть инструкция по подключению. Чтобы с ней
ознакомиться, нажмите справа «Подключение инструмента».

  1. Откройте [RuStore Консоль](https://console.rustore.ru/waiting).

  2. Перейдите на вкладку «Инструменты».

  3. Выберите приложение, к которому нужно подключить инструмент. Откроется полный список инструментов, доступных в RuStore.

![](https://www.rustore.ru//help/assets/images/tools-
catalog-a9fce65fc35dcb41d24fe201dd5541fe.png)

  4. Выберите инструмент, который нужно подключить для приложения.

  5. Нажмите «Подключить» на странице инструмента.

Инструмент подключится для выбранного приложения, и его статус изменится на
«Подключен».

Доступ к инструменту будет только у владельца организации в RuStore. Чтобы
дать доступ другим сотрудникам, перейдите в сервис и настройте доступ в
интерфейсе инструмента.
