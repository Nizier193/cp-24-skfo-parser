# Как вставить ссылку на приложение из RuStore в группе ВКонтакте

Настройте кнопку «Действия» в сообществе ВКонтакте, чтобы пользователи могли
скачивать ваше приложение напрямую из RuStore.

примечание

Изменение настроек доступно только для владельца или администратора
сообщества.

## Как настроить кнопку действия?​

  1. Откройте раздел «Управление» в сообществе.
  2. В блоке «Кнопка действия» измените статус на «Включена».
  3. В поле «Тип действия» выберите пункт «Открыть мобильное» приложение.
  4. Добавьте ссылку на приложение в поле RuStore.
  5. Нажмите «Сохранить», чтобы применить настройки.
