# Страница приложения в RuStore: Все о приложении в одном месте

Выберите в [интерфейсе RuStore](/help/users/start/interface) приложение и
нажмите на него.

Откроется окно с информацией о приложении:

  * иконка;
  * название;
  * разработчик;
  * возрастное ограничение;
  * размер приложения;
  * кнопка для установки/обновления приложения на мобильное устройство;
  * категории;
  * видео и скриншоты, доступные для смартфонов/планшетов;
  * полное описание приложения;
  * описание последней версии приложения;
  * история версий;
  * общий рейтинг приложения;
  * список последних отзывов к приложению.
