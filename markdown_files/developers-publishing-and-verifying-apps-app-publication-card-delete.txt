# Как удалить карточку приложения в RuStore: подробная инструкция

Вы можете удалить карточку приложения из [Консоли
RuStore](https://console.rustore.ru/sign-in). Удаление доступно только для тех
приложений, в которых ни одна из версий не была отправлена на модерацию.

примечание

Удалить карточку приложения может только владелец аккаунта — см. [Роли
пользователей в RuStore](/help/developers/developer-account/user-roles).

  1. Перейдите в раздел «Приложения».
  2. Выберите нужное приложение и нажмите ![icon](https://www.rustore.ru//help/three-dots-hor-icon.svg) чтобы перейти в контекстное меню.
  3. Выберите пункт «Удалить приложение».
  4. Нажмите на кнопку «Удалить».
