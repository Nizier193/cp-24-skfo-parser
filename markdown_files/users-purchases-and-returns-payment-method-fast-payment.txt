# Оплата через СБП: Удобные и быстрые платежи для приложений и товаров

Вы можете оплатить приложения или товары внутри приложения с помощью системы
быстрых платежей.

  1. Откройте приложение, в котором хотите оформить подписку или купить товар в приложении.
  2. Выберите необходимые условия.
  3. Нажмите «Оформить».
  4. Нажмите «Система быстрых платежей».
  5. Выберите банк для оплаты.

Вы перейдете в мобильное приложение банка для подтверждения и завершения
оплаты.
