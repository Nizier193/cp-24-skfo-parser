# Простая инструкция по выходу из аккаунта

  1. Нажмите ![icon](https://www.rustore.ru//help/rustore-icon.svg) на домашнем экране вашего устройства.
  2. Нажмите ![icon](https://www.rustore.ru//help/profile-icon.svg) в нижнем меню.
  3. Нажмите на название своего аккаунта.
  4. Нажмите «Выйти из аккаунта».
